# Grant Site Monitor Bot

Серверный бот для Telegram, который проверяет сайт грантов и рассылает подписчикам уведомление `Сайт ожил`, когда сайт переходит из `404/504` в рабочий статус.

## Как использовать

1. Создайте `.env` из примера:
   ```bash
   cp .env.example .env
   ```
2. Вставьте `BOT_TOKEN` вашего бота.
3. Запустите:
   ```bash
   docker compose up -d --build
   ```
4. Откройте вашего бота в Telegram и отправьте `/start`.

После этого любой грантополучатель может открыть бота и тоже отправить `/start`, чтобы подписаться.

## Команды бота

- `/start` подписаться на уведомления
- `/stop` отписаться
- `/status` посмотреть последний статус сайта

## Полезные команды

```bash
docker compose logs -f
docker compose down
```
